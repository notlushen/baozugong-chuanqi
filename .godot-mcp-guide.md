# Godot MCP 使用指南

## 可用工具

当需要操作 Godot 编辑器时，使用 `godot` MCP 服务器提供的工具。

### 场景创建
- `create_scene` - 创建新场景文件
- `add_node` - 添加节点到场景
- `save_scene` - 保存场景到磁盘

### 脚本操作
- `create_script` - 创建 GDScript 文件
- `attach_script` - 附加脚本到节点
- `edit_script` - 编辑脚本内容

### 项目信息
- `get_project_info` - 获取项目信息
- `get_filesystem_tree` - 获取文件系统树
- `get_scene_tree` - 获取场景树

### 运行调试
- `play_scene` - 运行场景
- `stop_scene` - 停止场景
- `get_editor_errors` - 获取编辑器错误

## 工作流程示例

### 创建 2D 玩家场景
1. `create_scene` root_type="Node2D" path="res://scenes/player.tscn"
2. `add_node` 添加 Sprite2D
3. `add_node` 添加 CollisionShape2D
4. `create_script` 编写玩家逻辑
5. `attach_script` 附加到根节点
6. `save_scene`
